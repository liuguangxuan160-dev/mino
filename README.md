# mino
# 求职资料整理助手  这是一个基于大模型 API 的求职辅助工具，主要用于整理岗位 JD、优化简历、总结项目经历和生成面试准备材料。  ## 计划功能  - JD 关键词和岗位要求提取 - 简历与岗位匹配度分析 - 针对不同岗位生成简历 bullet - 项目 README / 代码片段总结 - 根据 JD 和简历生成面试问题 - 保存不同岗位的分析记录  ## 技术栈  - Frontend: React / Next.js - Backend: Python FastAPI 或 Node.js - Model: MiMo API  ## 开发计划  第一阶段：完成 JD 分析和简历对比   第二阶段：完成简历改写和面试题生成   第三阶段：加入项目总结和历史记录功能

# 求职资料整理助手

这是一个基于大模型 API 的求职辅助工具，主要用于整理岗位 JD、优化简历、总结项目经历和生成面试准备材料。

我目前正在找工作，发现不同岗位的 JD 要求差异比较大，简历和项目经历也需要针对岗位反复调整。这个项目的目标是把这些重复但重要的工作整理成一个简单工具，提高投递和面试准备效率。

## 计划功能

- JD 关键词和岗位要求提取
- 简历与岗位匹配度分析
- 针对不同岗位生成简历 bullet
- 项目 README / 代码片段总结
- 根据 JD 和简历生成面试问题
- 保存不同岗位的分析记录

## 使用流程

1. 输入岗位 JD
2. 上传或粘贴个人简历
3. 系统提取岗位要求和关键词
4. 对比简历与岗位要求的差距
5. 生成简历修改建议和面试准备问题

## 技术栈

- Frontend: React / Next.js
- Backend: Python FastAPI 或 Node.js
- Model: MiMo API
